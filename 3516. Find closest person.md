## ðŸ§© Approach

- Find the **absolute difference** between `x` and `z`, and between `y` and `z`.
- Compare:
  - If `diff1 < diff2`, first person is closer â†’ return `1`.
  - If `diff1 > diff2`, second person is closer â†’ return `2`.
  - If both are **equal**, return `0`.

âœ… **Absolute difference** is used because distance cannot be negative.

## ðŸ’» Java Solution

```java
class Solution { 
    public int findClosest(int x, int y, int z) {
        int diff1 = Math.abs(x - z);
        int diff2 = Math.abs(y - z);
        
        if (diff1 < diff2) {
            return 1;
        } else if (diff1 == diff2) {
            return 0;
        } else {
            return 2;
        }
    }
}
```
