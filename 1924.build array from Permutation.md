### âœ… Java Solution:

```java
class Solution {
    public int[] buildArray(int[] nums) {
        int i = nums.length;
        int[] ans = new int[i];
        for (int x = 0; x < i; x++) {
            ans[x] = nums[nums[x]];
        }
        return ans;
    }
}
```
